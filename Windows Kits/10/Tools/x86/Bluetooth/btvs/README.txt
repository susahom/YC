Bluetooth Virtual Sniffer (btvs.exe)

The Bluetooth Virtual Sniffer allows the user to view live traces in the Frontline ComProbe Protocol Analysis System software. 

Prerequisites:
1. Have the Frontline ComProbe Protocol Analysis system (http://www.fte.com/) installed

Usage:
1. Start tool
      a. This will open ComProbe Protocol Analysis System
2. Click the record button (Red button on tool bar)
3. Click View->Frame Display
4. Start Bluetooth activity
